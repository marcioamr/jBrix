Exemplo jBrix
QR Code Dinâmico QR Code Estático

Receba por Pix

Utilize o plugin do Brix para jQuery e facilmente gere um Copia e Cola ou um QR Code para ser pago por qualquer PSP.

Utilizando

Para utilizar o componente basta baixá-lo e colocá-lo junto com os arquivos do seu site e referenciá-lo normalmente.

<script src="js/jquery.brix.js"></script>

jQuery

O plugin pode ser utilizado para gerar o Copia e Cola ou um QR Code. Para criar é necessário informar credencial, a chave Pix cadastrada em seu PSP de preferência e cadastrada na plataforma, o valor a ser recebido e uma descricao para o recebimento.

A credencial é uma informação obtida em seu cadastro na plataforma e permite o acesso à API de integração com o Pix.

Criação do QR Code e código Copia e Cola

Primeiramente é necessário definir a DIV que receberá o Copia e Cola e a que receberá a imagem do QR Code.

<div id="copiaecola"></div>
<div id="qrcode"></div>

Posteriormente inserir o trecho javascript abaixo referenciando as DIV criadas.

$(document).ready(function(){
    $('#copiaecola').brix({
        credencial: 'zh9evIvbMxpV4VWMSBIvUxdsUz0EEkTs4rbyW6HU',
        chave: 'ricardo@usejuntos.com.br',
        valor: 1.12,
        decricao: "Restaurante"
    }, function(copiaecola) {
        $('#qrcode').brix('qrcode', copiaecola)
    });
});
